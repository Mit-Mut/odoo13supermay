POS CASH DRAWER ALERT

POS Cash Drawer Alert module will help to set a Threshold limit for POS cash drawer and will provide an alert feature that notifies the POS user when the threshold limit is reached and also POS user can withdraw required amount from cash drawer in running session

Features
  • Threshold limit can be configured for the POS cash drawer.
  • Alert feature on touching threshold limit.
  • Seller can directly take out money from running session of POS
  • Makes it convenient for POS users for managing cash in POS cash drawer.
  • POS Session View & Configuration

POS CASH DRAWER ALERT

Configuration To Enable Cash Drawer Alert Feature

Enable cash control option and set cash drawer threshold limit so that alert message will be shown in POS session whenever cash drawer threshold limit is reached

Cash Drawer Alert in POS Session

This alert popup message appears when cash drawer reaches its threshold limit. A POS User can immediately take money out from cash drawer by clicking on Take money out button.

Withdraw Cash From Cash Drawer In Running Session

This popup message will appear to take money out from cash drawer. Click on proceed button to withdraw amount from cash drawer.
